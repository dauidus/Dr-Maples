2. HTML Structure

This theme is a fixed layout. Some pages, like "about.html" and "blog.html" with two columns, single text page "fullwidth.html" with 1 wide column. The general template structure is the same throughout the template. Here is the general structure:

<!--Header-->
<div id="header">
  Logo, slogan, rss-link, search form go here
  
  <!--Navigation-->
  <div id="nav">
    Navigation goes here
  </div>

</div>
<!--End Header-->

<!--Content-->
<div id="content">
  Content goes here
</div>

<!--Footer-->
<div id="footer">
  Footer goes here
</div>


Two columns pages (about.html, blog.html)
<!--Content left part-->
<div id="content_left">
  Content goes here
</div>

<!--Content right part-->
<div id="content_right">
  Sidebar content goes here
</div>

3. CSS Files and Structure

The first one is a general style file Styles.css
and there are 11 CSS files in the folder CSS - it's for color schemes.

The general file contains all of the specific stylings for the page. The file is separated into sections using:

	
/* =========== Header =========== */
some code
/* =========== End Header =========== */


/* =========== Navigation =========== */
some code
/* =========== End navigation =========== */


/* =========== Footer =========== */
some code
/* =========== End Footer =========== */


/* =========== Index Articles =========== */
some code
/* =========== End Index Articles =========== */

some code
	
etc, etc.

If you would like to change the color scheme of website, find this code in your HTML file's head:

<!--CSS Color Scheme-->
<link rel="stylesheet" href="css/bamboo.css" type="text/css" media="screen" />
And replace "css/bamboo.css" to another CSS file, for example to the "css/blue.css" or "css/black.css".

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

	
#footer a {
  color: #someColor;
}

This theme imports some Javascript files. They are in the folder named "js".

  2. Typeface
    With typeface.js you can embed custom fonts in your web pages so you don't have to render text to images.
    More information about using Typefase

  3. Portfolio zoom icon effect
    Very simple script in the folder js/zoom/

  4. Superfish Dropdown menu
    Superfish is a nice jQuery plugin that takes an existing pure CSS drop-down menu.

  5. Fancybox (for Portfolio page)
    FancyBox is a tool for displaying images, html content and multi-media in a Mac-style "lightbox" that floats overtop of web page.
